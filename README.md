# Presence App

A presence mobile app using getX and firebase.

## Getting Started

This project build with :

- Firebase Auth
- Firebase Storage
- FireStore
- FlutterFire
- GetX CLI
- GetX State Management
- Geolocator

Feature
- Login with Email and Password
- Add user from admin 
- Email Verification
- Forgot Password
- Change Password
- Update Profile
- ETC
